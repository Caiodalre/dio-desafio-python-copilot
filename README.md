# Desafio Python com GitHub Copilot - DIO

Repositório criado para o desafio de projeto da DIO, com foco na utilização do GitHub Copilot como ferramenta de apoio no desenvolvimento de algoritmos em Python.

## Objetivo

O objetivo deste projeto é praticar a resolução de problemas utilizando Python, aplicando conceitos de lógica de programação, entrada e saída de dados, manipulação de strings e organização de código.

## Tecnologias utilizadas

- Python
- Git
- GitHub
- GitHub Copilot
- Markdown

```text

## Desafios desenvolvidos

### Desafio 01 - Clonar repositório

Recebe uma URL de repositório GitHub e retorna o comando Git necessário para clonar o projeto.

### Desafio 02 - Adicionar arquivos

Recebe o nome de um arquivo ou padrão de arquivos e retorna o comando Git necessário para adicioná-los à área de preparação.

### Desafio 03 - Criar branch

Recebe o nome de uma nova branch e retorna o comando Git necessário para criá-la.

### Desafio 04 - Trocar de branch

Recebe o nome de uma branch existente e retorna o comando Git necessário para acessá-la.

### Desafio 05 - Mesclar branch

Recebe o nome de uma branch e retorna o comando Git necessário para mesclá-la com a branch atual.

## Aprendizados

Durante este desafio, pratiquei a criação de soluções simples em Python e utilizei o GitHub Copilot como ferramenta de apoio para acelerar o desenvolvimento, sugerir estruturas de código e melhorar a produtividade.

Também reforcei conceitos importantes de versionamento de código, documentação de projetos e organização de repositórios no GitHub.

```
